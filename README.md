# dusk-resume-template

一个基于 Vue 3 和 ElementPlus 的简历模板。

直接修改 /src/data.js 里的数据就可以修改简历资料。

## 截图
![截图](https://vkceyugu.cdn.bspapp.com/VKCEYUGU-6fec5402-1210-4b81-9949-97c1987784b9/378a4378-ed6b-4003-ab6c-f1bb81a3ad12.png)

 ## 开发计划
 - [ ] 线上直接修改配置文件生成新简历
 - [ ] 把服务部署到云服务上成为公开应用

## 安装

```
yarn install
```

### 运行调试
```
yarn serve
```

### 打包
```
yarn build
```
